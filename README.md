##  Overview
This is my **personal portfolio website** built with HTML, CSS, and JavaScript.  
It highlights my projects, skills, and achievements with a clean, responsive design and interactive sections like Home, About, Projects, and Contact.  
It provides an engaging professional online presence.

##  Features
- Fully **responsive design** for mobile and desktop
- Clean and modern layout with easy navigation
- **Projects section** with descriptions and visuals
- Smooth scrolling and interactive UI elements
- **Contact section** to connect with me
